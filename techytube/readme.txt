== Description ==

This plugin allows you to embed youtube videos on your blog.
The code of embeded videos is a valid XHTML Markup and compatiable with all major web browsers. This plugin can come handy to all those who embed many videos their blogs.
Embeded videos are in 425px by 344px in dimension.

Usage

* While writing/editing your post
* Wrap the youtube video id between [techytube][/techytube] tags embed a youtube video.
You can get the youtube Video ID from the URL of the video
http://www.youtube.com/watch?v={video_id_here}
 (Please note that [techytube][/techytube] are case sensitive)
* Anything between [techytube][/techytube] will not be visible directly on your blog.
* Example : This is the normal post content [techytube]_OBlgSz8sSM[/techytube] will embed the video from this page
http://www.youtube.com/watch?v=_OBlgSz8sSM

== Installation ==

Installation is simple:

* Download the plugin.
* Copy `techytube` folder to the plugins directory of your blog.
* Enable the techytube-xhtml plugin in your admin panel.
